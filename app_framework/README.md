# App Framework.
This is a starter application framework built in Odin. It scaffolds a basic UI application and allows comsumers to build custom UI's easily on top.

## Libraries 
This package uses the following libraries:
* SDL 3: Windowing/Input
* OpenGL: GPU Accelerated UI drawing
* NanoVG: Drawing library that uses OpenGL to draw elements
* Clay: UI Layout library for implementing css style flexbox layouts
